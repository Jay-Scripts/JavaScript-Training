# JavaScript Self-Training 

A personal **JavaScript self-training repository** designed to improve your coding skills through structured exercises, practice projects, and mini challenges.  

---

##  Overview

This repository contains notes, exercises, and projects that cover **core JavaScript concepts** from basics to advanced topics. It is intended for **self-learners** who want a practical, hands-on approach to mastering JavaScript.

---

##  Topics Covered

- JavaScript Basics: variables, data types, operators  
- Functions and Scope  
- Conditional Statements and Loops  
- Arrays and Objects  
- DOM Manipulation  
- Event Handling  
- ES6+ Features: arrow functions, template literals, destructuring, spread/rest operators  
- Fetch API & AJAX  
- Local Storage  
- Mini Projects and Exercises for practice  

---

##  Tech Stack

- JavaScript (ES6+)  
- HTML5 & CSS3
