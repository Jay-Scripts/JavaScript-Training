Objective:
Learn the differences between let, var, and const and understand their scope (block vs function vs global).

Instructions:

Create a simple HTML page with:

A title: "JS Variables & Scope Activity"

"Test Let" → ID: btnLet

"Test Var" → ID: btnVar

"Test Const" → ID: btnConst

A paragraph where the results will appear → ID: output

JavaScript tasks:

For let:

Declare a variable x.

Try to redeclare x in the same scope (should fail).

Declare x inside a block and display the value.

For var:

Declare a variable y.

Redeclare y in the same scope (should succeed).

Declare y inside a block and display the value.

For const:

Declare a constant z.

Try to reassign z (should fail).

Declare z inside a block and display the value.

Use document.getElementById() to:

Attach click events to buttons

Show results in the paragraph

Add comments explaining each step.

Expected Learning Outcomes:

Understand block vs function/global scope

See which variables can be redeclared or reassigned

Practice using document.getElementById()
